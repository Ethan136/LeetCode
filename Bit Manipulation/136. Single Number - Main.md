# LeetCode
[136. Single Number](https://leetcode.com/problems/single-number/)

# Answer
```Cpp
class Solution {
public:
    int singleNumber(vector<int>& nums) {
        int result = 0;
        for(int num : nums) result ^= num;
        return result;
    }
};
```
- XOR: 相同 bit 變成 0，不同 bit 變成 1
- A ^ 0 -> A
- A ^ A -> 0
- 所有數字都是2次，只有一個1次
	- 出現2次的數字，XOR 後會相消變成 0
	- 出現1次的數字，和 0 XOR 後會維持這個數字
