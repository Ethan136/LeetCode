# LeetCode
[191. Number of 1 Bits](https://leetcode.com/problems/number-of-1-bits/)

# Answer
```Cpp
class Solution {
public:
    int hammingWeight(int n) {
        int bitCnt = 0;
        while(n) {
            if(n & 1) bitCnt++;
            n = (n >> 1);
        }
        return bitCnt;
    }
};
```
- 每次都檢查 bit 0，檢查後把原始數字 shift right 1 bit
- 一旦原始數字為 0 -> 停止檢查
