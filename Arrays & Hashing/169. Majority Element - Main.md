# LeetCode
[169. Majority Element](https://leetcode.com/problems/majority-element/)

# Answer 1st
## Brute Force
```Cpp
class Solution {
public:
    int majorityElement(vector<int>& nums) {
        unordered_map<int, int> numToCnt;

        int result;
        int maxDupCnt = 0;
        for(int num : nums) {
            numToCnt[num]++;
            if(numToCnt[num] <= maxDupCnt) continue;

            result = num;
            maxDupCnt = numToCnt[num];
        }
        return result;
    }
};
```

## Optimal - 依照題目特性劃分
- 題目有說『max cnt num 一定會出現超過一半次數』
- 所有可以把『max cnt num 分一羣』以及『other num 分一羣』，再用 relative cnt 來找出 max cnt num
```Cpp
class Solution {
public:
    int majorityElement(vector<int>& nums) {

		int relativeMaxCnt = 0;
        int maxCntNum;
        for(int num : nums) {
            if(relativeMaxCnt == 0) {
                relativeMaxCnt = 1;
                maxCntNum = num;
            }
            else {
                relativeMaxCnt += (num == maxCntNum) ? (1) : (-1);
            }
        }
        return maxCntNum;
    }
};
```
