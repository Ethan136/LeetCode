# 想法筆記
- 使用 linear search 硬幹會超過等待時間

# Solution 1: Hash Set Approach
```C++
class Solution {
public:
	bool containsDuplicate(vector<int>& nums) {
		for(int i = 1; i < nums.size(); i++) {
			for(int j = 0; j < i; j++) {
				if(nums[j] == nums[i]) return true;
			}
		}
		return false;
	}
};
```
